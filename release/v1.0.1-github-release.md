# v1.0.1: New-User Release Polish

`one-person-company-os` v1.0.1 tightens the public surface for first-time users and keeps the project language aligned around the visual operating cockpit.

Highlights:

- README and ClawHub listing now lead with the founder value, workspace outputs, and first-run promise
- English and Chinese guides now describe the operating loop directly
- bundled playbooks now use current goal, bottleneck, primary arena, and shortest action language
- sample materials now point to the operating cockpit and visual assets
- release validation now checks public docs and standalone role brief output for stale visible workflow language

The safety boundary remains unchanged: no automatic system package installation, no required API keys for normal use, and all generated files stay inside the founder-approved workspace.
