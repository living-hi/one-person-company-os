# v1.0.2: Multi-Platform Adapter Pack

`one-person-company-os` v1.0.2 adds a cross-market publishing layer for the visual operating cockpit.

Highlights:

- Claude Skills adapter
- Hermes Agent adapter with MCP config
- stdio MCP server for MCP-compatible clients
- OpenAI GPT Store instructions and optional Actions schema
- Dify plugin submission draft
- Poe bot prompt package
- Gemini Gem instruction package
- GitHub Copilot Extension planning manifest
- Microsoft Copilot Studio agent draft
- platform publishing status, requirements, and experience log
- platform adapter validation wired into release validation

The core runtime and safety boundary remain unchanged: no automatic system package installation, no unrelated credentials for normal use, and all generated files stay inside the founder-approved workspace.
