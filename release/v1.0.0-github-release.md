# v1.0.0: Visual Operating Cockpit

`one-person-company-os` v1.0 is a breaking architecture release.

The system is now centered on a visual operating cockpit for AI-native one-person companies:

- v1.0 business-loop state without old `stage_id` or `current_round` fields
- localized HTML cockpit entry after download
- deterministic SVG visuals for the business loop and revenue pipeline
- optional AI image prompts for launch and promotional assets
- old stage/round commands now return deprecation guidance

The core loop stays:

`promise -> buyer -> product capability -> delivery -> cash -> learning -> asset`

This release keeps the safety boundary intact: no automatic system package installation, no required API keys for normal use, and all generated files stay inside the founder-approved workspace.
