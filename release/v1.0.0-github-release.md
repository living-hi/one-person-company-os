# v1.0.0: Visual Operating Cockpit

`one-person-company-os` v1.0.0 centers the founder experience on a visual operating cockpit for AI-native one-person companies.

The core loop is:

`promise -> buyer -> product capability -> delivery -> cash -> learning -> asset`

Highlights:

- localized HTML cockpit entry after download
- deterministic SVG visuals for the business loop and revenue pipeline
- optional AI image prompts for launch and promotional assets
- role briefs, reminders, flow notes, and DOCX starter templates aligned to the operating loop
- no automatic system package installation
- no required API keys for normal use
- all generated files stay inside the founder-approved workspace
