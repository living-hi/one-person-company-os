# Claude Skills Install Notes

Copy the repository package into a Claude Skills location supported by the user's Claude environment.

Recommended folder name:

```text
one-person-company-os/
```

Minimum files:

```text
SKILL.md
scripts/
references/
assets/
agents/
orchestration/
```

For Claude environments that do not allow local script execution, install only `platforms/claude-skill/SKILL.md` as the prompt skill and use the canonical repository as downloadable reference material.
