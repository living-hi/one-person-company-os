# agentskills.io Listing

Name: One Person Company OS

Version: 1.0.3

Category: startups, productivity, AI agents, operating system

Summary:

Build and advance a visual operating cockpit for an AI-native one-person company.

Description:

One Person Company OS helps a solo founder turn an AI product idea into a visible operating loop:

`promise -> buyer -> product capability -> delivery -> cash -> learning -> asset`

It focuses the founder on the current primary goal, current bottleneck, primary arena, and shortest action today. The Hermes Agent package includes a skill, SOUL notes, and MCP configuration for local execution.

First run:

- confirm the sellable promise
- identify the first buyer
- clarify the core problem
- name the current primary goal
- name the current bottleneck
- choose the primary arena
- choose the shortest action today
- confirm the founder-approved workspace path

Safety boundary:

- Normal use does not require unrelated credentials.
- Persistent files stay inside the founder-approved workspace.
- The skill does not auto-install system packages.
- Pricing, legal, launch, budget, and risky customer-facing actions require founder approval.
- AI images are optional creative assets and must not carry exact operating numbers.

Local package:

- `platforms/hermes-agent/SKILL.md`
- `platforms/hermes-agent/SOUL.md`
- `platforms/hermes-agent/mcp-config.json`
- `platforms/hermes-agent/install.md`
- `platforms/mcp-server/server.py`
